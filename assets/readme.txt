convert.bat uses the 1bpp2c.exe that is in it's current directory or path environment variable
to convert the any (1-bit per pixel) .bmp files in the current folder.

the executables 1bpp2c.exe are located in the "..\bin" under the folder for the appropriate architecture
..\x16 for legacy MSDOS environment and Windows 9x.
..\x32 for windows 7 and above.
..\x64 for 64-bit Windows 7 and above.
